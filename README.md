### Membuka Website
1. Buka file `public/index.html` di browser
2. Navigasi menggunakan navbar ke halaman yang diinginkan

### Login Admin
1. Akses: `public/pages/admin.html`
2. **Username**: `admin`
3. **Password**: `nawasena2026`

### Mengelola Konten

#### Tambah Artikel
1. Login ke admin panel
2. Klik tab "Artikel"
3. Klik tombol "+ Tambah Artikel"
4. Isi form (judul, ringkasan, konten, kategori, penulis, gambar)
5. Klik "Simpan"

#### Edit/Hapus Artikel
1. Pada tabel artikel, klik tombol "Edit" atau "Hapus"
2. Untuk edit: ubah data lalu klik "Simpan"
3. Untuk hapus: konfirmasi penghapusan

#### Kelola Post
- Sama seperti artikel, gunakan tab "Post" di admin panel
